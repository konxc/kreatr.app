## 2. Jenis Data yang Kami Kumpulkan

Kami hanya mengumpulkan data yang diperlukan untuk memberikan layanan terbaik kepada Anda. Berikut rinciannya:

### 2.1 Data Akun dan Profil
Saat Anda mendaftar, kami mengumpulkan:
- **Informasi dasar**: Nama, email, nomor telepon (opsional)
- **Informasi profil**: Username, foto profil, bio
- **Informasi bisnis**: Nama bisnis/brand, industri, ukuran tim (untuk agency/UMKM)
- **Kredensial login**: Password terenkripsi, token autentikasi

### 2.2 Data dari Integrasi TikTok
Ketika Anda menghubungkan akun TikTok, kami mengakses:
- **Data profil TikTok**: Username, follower count, following count, bio
- **Statistik konten**: Jumlah views, likes, comments, shares dari video Anda
- **Metadata konten**: Caption, hashtag, musik yang digunakan, waktu posting
- **Data engagement**: Demografi audiens (usia, gender, lokasi), waktu terbaik posting
- **Performance metrics**: Engagement rate, growth rate, trending topics

**Catatan penting**: 
- Kami TIDAK mengakses Direct Messages (DM) Anda
- Kami TIDAK menyimpan video Anda (hanya metadata)
- Kami TIDAK posting atas nama Anda tanpa izin eksplisit

### 2.3 Data dari Integrasi Google Calendar
Untuk fitur penjadwalan konten, kami mengakses:
- **Informasi kalender**: Nama kalender, zona waktu
- **Event scheduling**: Tanggal dan waktu posting yang dijadwalkan
- **Permission**: Hak akses untuk membuat dan mengedit event

**Catatan penting**:
- Kami TIDAK membaca event pribadi Anda yang tidak terkait dengan kreatr.app
- Kami hanya membuat/edit event yang Anda buat melalui platform kami

### 2.4 Data Aktivitas di Platform
Kami mencatat aktivitas Anda untuk meningkatkan pengalaman:
- **Log aktivitas**: Fitur yang digunakan, waktu akses, durasi sesi
- **Riwayat transaksi**: Pembelian kredit, penggunaan layanan SMM
- **Konten yang dibuat**: Ide konten hasil brainstorming AI, draft caption, hashtag research
- **Pengaturan preferensi**: Bahasa, notifikasi, tema tampilan
- **Feedback**: Rating, review, laporan bug

### 2.5 Data Kolaborasi Tim
Untuk fitur collaboration (seperti Notion/Figma), kami menyimpan:
- **Struktur tim**: Nama tim, role member (owner, editor, viewer)
- **Shared content**: Konten yang dibagikan antar anggota tim
- **Activity log**: Siapa mengedit apa dan kapan
- **Komunikasi tim**: Komentar, mention, notifikasi internal

### 2.6 Data Teknis dan Otomatis
Sistem kami secara otomatis mengumpulkan:
- **Informasi perangkat**: Jenis device, sistem operasi, browser
- **Data jaringan**: Alamat IP, ISP, lokasi geografis (negara/kota)
- **Cookie dan tracking**: Identifier unik untuk session management
- **Analytics**: Heatmap, click patterns, scroll depth (untuk UX improvement)

### 2.7 Data Pembayaran
Untuk transaksi pembelian kredit:
- **Informasi pembayaran**: Metode pembayaran (e-wallet, bank transfer, kartu kredit)
- **Riwayat transaksi**: Tanggal, jumlah, status pembayaran
- **Invoice**: Bukti pembayaran elektronik

**Catatan penting**:
- Kami TIDAK menyimpan detail kartu kredit secara langsung
- Pemrosesan pembayaran dilakukan melalui payment gateway yang aman dan tersertifikasi PCI-DSS

## 4. Penyimpanan dan Keamanan Data

### 4.1 Lokasi Penyimpanan Data
- Data disimpan di server cloud yang berlokasi di **Indonesia dan Singapura**
- Kami menggunakan infrastruktur cloud dari provider terpercaya (AWS/GCP/Azure)
- Backup data dilakukan secara rutin untuk disaster recovery

### 4.2 Durasi Penyimpanan
- **Data aktif**: Selama akun Anda aktif
- **Data transaksi**: Minimum 5 tahun (sesuai regulasi perpajakan)
- **Data marketing**: Hingga Anda unsubscribe atau menarik consent
- **Data setelah penghapusan akun**: 30 hari (untuk recovery), kemudian dihapus permanen

### 4.3 Enkripsi dan Keamanan
Kami menerapkan multiple layers of security:
- **Enkripsi data in-transit**: TLS/SSL 1.3 untuk semua komunikasi
- **Enkripsi data at-rest**: AES-256 untuk data sensitif di database
- **Token encryption**: OAuth tokens dan API keys terenkripsi
- **Password hashing**: Bcrypt dengan salt untuk password pengguna
- **Access control**: Role-based access control (RBAC) untuk tim internal

### 4.4 Praktik Keamanan Lainnya
- **Regular security audits**: Penetration testing dan vulnerability scanning
- **Firewall dan intrusion detection**: Melindungi dari serangan cyber
- **Monitoring 24/7**: Deteksi dan respons cepat terhadap insiden keamanan
- **Employee training**: Tim internal terlatih dalam praktik keamanan data
- **Data minimization**: Hanya menyimpan data yang benar-benar diperlukan

### 4.5 Apa yang Kami Lakukan Jika Terjadi Pelanggaran Data
Jika terjadi data breach yang berisiko terhadap privasi Anda:
- Kami akan **memberitahu Anda dalam 3x24 jam**
- Melaporkan kepada otoritas berwenang (Kominfo/BSSN) sesuai regulasi
- Mengambil langkah mitigasi untuk mencegah kerusakan lebih lanjut
- Memberikan panduan langkah yang harus Anda ambil (misal: ganti password)

## 5. Cookie dan Alat Analitik

### 5.1 Apa Itu Cookie?
Cookie adalah file kecil yang disimpan di browser Anda untuk mengingat preferensi dan aktivitas Anda. Kami menggunakan cookie untuk meningkatkan pengalaman Anda di platform.

### 5.2 Jenis Cookie yang Kami Gunakan

#### Essential Cookies (Wajib)
- **Session cookies**: Untuk menjaga Anda tetap login
- **Security cookies**: Untuk mencegah CSRF dan serangan lainnya
- Cookie ini tidak dapat dinonaktifkan karena diperlukan untuk fungsi dasar platform

#### Analytics Cookies (Opsional)
- **Google Analytics**: Untuk memahami traffic dan user behavior
- **Mixpanel/Amplitude**: Untuk product analytics
- **Hotjar**: Untuk heatmap dan session recording (anonim)

#### Marketing Cookies (Opsional)
- **Facebook Pixel**: Untuk retargeting ads
- **Google Ads**: Untuk conversion tracking
- **LinkedIn Insight**: Untuk B2B marketing

### 5.3 Cara Mengelola Cookie
Anda dapat mengatur preferensi cookie melalui:
- **Cookie consent banner** saat pertama kali mengunjungi website
- **Pengaturan akun** di dashboard kreatr.app
- **Browser settings** untuk memblokir semua cookie (dapat mempengaruhi fungsionalitas)

### 5.4 Alat Analitik Pihak Ketiga
Kami menggunakan:
- **Google Analytics**: Traffic analysis dan user demographics
- **Sentry**: Error monitoring dan crash reporting
- **LogRocket**: Session replay untuk debugging (dengan consent)

Data dari alat ini bersifat agregat dan dianonimkan sebisa mungkin.
