# @kreatr/ui

Shared UI components and utilities for kreatr.app

## Usage

```typescript
import { cn } from '@kreatr/ui'

// Use the cn utility to merge classes
const className = cn('base-class', 'additional-class', {
  'conditional-class': condition
})
```

## Components

Components will be added as needed. This package provides:

- Utility functions (cn for class merging)
- Shared UI components (to be added)
- Common types and interfaces

## Development

```bash
# Type check
bun run type-check

# Lint
bun run lint
```
