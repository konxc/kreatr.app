# Changelog

All notable changes to kreatr.app will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial project setup with monorepo structure
- Landing page with hero, features, pricing, and testimonials sections
- Complete database schema with Prisma
- tRPC API routers for auth, content, workspace, scheduler, analytics, and credits
- AI integration with OpenAI (GPT-4, DALL-E) and Anthropic Claude
- Social media integrations (TikTok, Instagram, Twitter/X)
- Google Calendar integration for content scheduling
- Workspace collaboration features with role-based access control
- Credit system for AI resource management
- Content generation with AI (ideas, captions, hashtags, voice-over)
- Analytics dashboard with performance insights
- Content scheduler with calendar view
- Comprehensive documentation (README, SETUP-GUIDE, API docs, CONTRIBUTING)
- CI/CD pipeline with GitHub Actions
- Docker setup for local development

### Security
- NextAuth.js authentication setup
- Environment variable management
- CORS and security headers configuration
- Rate limiting preparation

## [0.1.0] - 2025-01-XX

### Added
- MVP release
- Core features for content creation and scheduling
- Basic analytics and insights
- Free tier with limited features
- Boost and Pro subscription plans
